- Python: Version 3.8 or newer is recommended

- The files with the scripts need to be inside the same folder in the project directory.

- The following packages are necessary: 
    numpy
    torch
    matplotlib
sci-kit learn can be installed as well, in case we want to visualize a confusion matrix for the last case.

- To run the notebook:
    Open Visual Studio Code or Jupyter Notebook.
    Open the file "pathfinding.ipynb".
    Run the notebook cells in order.
